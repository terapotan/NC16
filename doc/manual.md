　本ドキュメントはNC-16の操作マニュアルです。
　Logisim-evolutionやcustomasmの導入が済んでいない方は、READMEに記載されている方法にしたがって、導入を済ませてください。

　



## 動作方法
　以下コマンドでアセンブリファイルをアセンブルすることができます。
```
customasm main.asm
```
　リポジトリ内にあるnc16.circが回路本体となります。ファイルをダブルクリックして開きます。上バーメニューから指のアイコンをクリックした後、RAMを右クリックするとポップアップメニューが表示され、イメージのロードと書かれている項目を押します。前手順でアセンブルした機械語ファイルを選択します。
　イメージのロード完了後、RESETボタンを押し、Ctrl+Kを押してクロックをスタートさせると、RAMにロードされたプログラムが実行されます。
